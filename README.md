**Importante:** Es necesario iniciar cada proceso en una terminal independiente

## Iniciar servidor/socket
Ir al directorio de servidor
`cd server`
Instalar dependencias
`npm install`
Iniciar servidor (por defecto en puerto 4001)
`npm run serve`


## Iniciar app
Ir al directorio de servidor
`cd client`
Instalar dependencias
`npm install`
Iniciar app
`npm run start`
